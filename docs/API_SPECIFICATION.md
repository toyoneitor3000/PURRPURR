# API Specification - Pigmento Stickers\n\n## 1. Overview\nThis API handles the catalog, pricing engine, and order management for the Pigmento Stickers platform. It is designed to be consumed by the Next.js frontend and potentially mobile apps.\n\n## 2. Data Models\n\n### Product (Base SKU)\nRepresents a type of sticker product (e.g., Die Cut, Kiss Cut).\n```typescript\ninterface Product {\n  id: string;\n  name: string;\n  slug: string;\n  description: string;\n  base_price_per_sq_in: number;\n  min_quantity: number;\n  image_url: string;\n  tags: string[];\n}\n```\n\n### Quote (Pricing Calculation)\nDynamic pricing based on user configuration.\n```typescript\ninterface QuoteRequest {\n  product_id: string;\n  width_inch: number;\n  height_inch: number;\n  quantity: number;\n  material_id: string;\n}\n```\n\n## 3. Endpoints\n\n### Catalog\n- `GET /api/products`: List all available sticker types.\n- `GET /api/products/:slug`: Get details for a specific product.\n\n### Pricing Engine\n- `POST /api/quote`: Calculates the total price and unit price based on dimensions and breaks.\n\n### Orders (Future)\n- `POST /api/orders`: Create a new order.\n\n## 4. Pricing Algorithm Strategy\nPrice = (Area * BasePrice * MaterialFactor * Quantity) * DiscountTier\n\n- **Area**: Width * Height (in inches).\n- **MaterialFactor**: Vinyl (1.0), Holographic (1.5), Clear (1.2).\n- **DiscountTier**: 50+ units (5% off), 100+ (10% off), 500+ (20% off).
