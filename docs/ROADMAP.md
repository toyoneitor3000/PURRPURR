# Roadmap Técnico: PigmentosTkts

## Fase 1: MVP (Weeks 1-2)
- [ ] Configuración del repositorio y CI/CD.
- [ ] Diseño de Base de Datos (Schema de Eventos y Tickets).
- [ ] Autenticación de usuarios (Login/Register).
- [ ] Landing Page principal (Home).
- [ ] Detalle de Evento.

## Fase 2: Core Transactional (Weeks 3-4)
- [ ] Carrito de compras / Selección de entradas.
- [ ] Integración pasarela de pagos (Checkout).
- [ ] Generación de Tickets QR.
- [ ] Panel de usuario (Mis entradas).

## Fase 3: Admin & Ops (Weeks 5-6)
- [ ] Dashboard para organizadores (crear eventos, ver ventas).
- [ ] Scanner App (PWA o Mobile) para validación de QR en puerta.
- [ ] Reportes financieros.
