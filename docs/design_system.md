# Sistema de Diseño para Landing Page de Fotografía Automotriz

Este documento define el sistema de diseño para la landing page de fotografía automotriz, siguiendo los lineamientos de la marca Purpur.

## 1. Paleta de Colores

Inspirada en la elegancia automotriz y un toque distintivo Purpur.

- **Primario (Purpur Oscuro)**: #2A003A (Un púrpura muy oscuro, casi negro, para fondos y elementos principales).
- **Secundario (Gris Carbón)**: #1A1A1A (Para texto, fondos secundarios, y contrastes sutiles).
- **Acento (Púrpura Vibrante)**: #9D00FF (Para CTAs, iconos, y elementos interactivos que requieren atención).
- **Neutro Claro (Blanco Suave)**: #F8F8F8 (Para texto en fondos oscuros y secciones claras).
- **Neutro Oscuro (Gris Medio)**: #808080 (Para texto secundario o bordes sutiles).

## 2. Tipografía

Una combinación de fuentes modernas y legibles para una experiencia visual impactante.

- **Encabezados (Display)**: 'Montserrat' (Sans-serif moderna, audaz y versátil para títulos).
  - `h1`: 48px, Bold
  - `h2`: 36px, Semi-Bold
  - `h3`: 28px, Medium
- **Cuerpo de Texto (Body)**: 'Roboto' (Sans-serif, limpia y altamente legible para párrafos y descripciones).
  - `p`: 16px, Regular
  - `small`: 14px, Regular

## 3. Espaciado (Spacing)

Definición de una escala de espaciado basada en múltiplos de 8px para consistencia.

- `spacing-xs`: 4px
- `spacing-sm`: 8px
- `spacing-md`: 16px
- `spacing-lg`: 24px
- `spacing-xl`: 32px
- `spacing-xxl`: 48px

## 4. Componentes Maestros (Ejemplos)

### a. Hero Section

- **Descripción**: La primera sección visible, captura la atención del usuario.
- **Elementos**: Imagen/video de fondo de automóvil, título impactante, subtítulo, botón CTA principal.
- **Comportamientos/Estados**:
  - **Hover en CTA**: Ligero cambio de color o elevación.
  - **Responsive**: Ajuste de tamaño de texto e imagen para móviles.

### b. Navigation Bar

- **Descripción**: Permite la navegación principal del sitio.
- **Elementos**: Logo de la marca, enlaces de navegación (Inicio, Servicios, Portafolio, Contacto).
- **Comportamientos/Estados**:
  - **Scroll**: Puede volverse pegajosa o cambiar de color al hacer scroll.
  - **Activo**: Resaltado del enlace de la página actual.
  - **Mobile**: Menú hamburguesa para pantallas pequeñas.

### c. Gallery / Portafolio

- **Descripción**: Muestra ejemplos de trabajos de fotografía automotriz.
- **Elementos**: Miniaturas de imágenes, filtros por tipo de vehículo/estilo.
- **Comportamientos/Estados**:
  - **Hover en miniatura**: Efecto de superposición o zoom sutil.
  - **Click en miniatura**: Abre una lightbox o vista detallada de la imagen.
  - **Filtrado**: Actualización dinámica de las imágenes mostradas.

### d. Call-to-Action (CTA) Section

- **Descripción**: Sección para incentivar al usuario a contactar o solicitar un servicio.
- **Elementos**: Título persuasivo, texto descriptivo, botón CTA secundario (ej. "Solicitar Cotización").
- **Comportamientos/Estados**:
  - **Hover en botón**: Similar al CTA principal.

### e. Footer

- **Descripción**: Contiene información de contacto, enlaces legales y redes sociales.
- **Elementos**: Logo, enlaces rápidos, información de contacto, iconos de redes sociales, aviso de derechos de autor.
- **Comportamientos/Estados**:
  - **Enlaces**: Cambio de color al hacer hover.
